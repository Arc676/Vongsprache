# Auf Vongsprache geschriebene Beispielsprogramme

## Fakultät

```
i bims fakultaet vong Funktionigkeit mit (n)
	bims n groesser 1 vong Wahrigkeit
		hab n mal bidde fakultaet mit (n minus 1)
	her
	hab n
her
bidde drucke mit (
	bidde fakultaet mit (
		bidde zuZahl mit (
			bidde gib mit ("Geben Sie eine Zahl ein")
		)
	)
)
```

## Sterne

```
i bims Anzahl vong bidde gib mit ("Wie viele Sterne möchten Sie?") her
Anzahl bimst bidde zuZahl mit (Anzahl)
mit j vong 0 bis Anzahl
	bidde drucke mit ("*")
her
```
